# Overview

The `HotelRatePlanInventoryNotif` message contains information about the inventory setup information that should be followed by the structure: Hotel > Rate > Room.

:::note
EXPLICAR AQUÍ QUE EXISTE LA FULL COPY Y LAS DELTAS
:::

Tenemos un timeout de 5000ms, si no respondes en ese tiempo nada, no volvemos a intentar enviar una request.

:::caution

If you do not get information for a certain range of dates that you got previously loaded in the system, do not overwrite them, as this means these days conditions that don’t appear in the request have not changed.

:::
