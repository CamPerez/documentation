# Overview

You can receive instant API notifications from TravelgateX with the `HotelResNotif` message for bookings and the `CancelNotif` message for cancellations. 

Also, it is possible to implement `HotelResRetrieve` messages to obtain a list of bookings from the Inventory Tool System.

:::note

Notice that card payments are only allowed if you implement `HotelResNotif` message

:::
