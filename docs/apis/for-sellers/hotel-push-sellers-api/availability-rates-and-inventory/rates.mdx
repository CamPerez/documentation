# Rates

## Rate Plans Push

Method to **push** rate plans to the product.

### Request

The `HotelRatePlanNotif` request message pushes rate plans for the [inventory](./inventory) that has been previously set up.

[Añadir ejemplo]

### Response

After each request, Travelgate processes the data and provides a response to the Seller. Upon receiving a `HotelRatePlanNotif` request, Travelgate sends a corresponding `HotelRatePlanNotif` response. The response options include either **success** or an **error**.

**Success:**

```html
<HotelRatePlanNotifResponse xmlns="http://schemas.xmltravelgate.com/hubpush/provider/2012/10">
    <HotelRatePlanNotifResult>
        <Success xmlns="http://www.opentravel.org/OTA/2003/05"/>
    </HotelRatePlanNotifResult>
</HotelRatePlanNotifResponse>
```

**Error:**

```html
<HotelRatePlanNotifResponse xmlns="http://schemas.xmltravelgate.com/hubpush/provider/2012/10">
    <HotelRatePlanNotifResult>
        <Errors xmlns="http://www.opentravel.org/OTA/2003/05">
            <Error ShortText="Incomplete AdditionalGuestAmount values" Code="7"/>
        </Errors>
    </HotelRatePlanNotifResult>
</HotelRatePlanNotifResponse>
```

## Rate Plans Retrieval

Method to **retrieve** the rate plans of the product.

### Request

The `HotelRatePlanRetrieve` request message is sent to retrieve a complete breakdown of rate plans, including details about the hotel, rate plans, and rooms.

[Añadir ejemplo]

### Response

After each request, Travelgate processes the data and provides a response to the Seller. Upon receiving a `HotelRatePlanRetrieve` request, Travelgate sends a corresponding `HotelRatePlanRetrieve` response. 

The response options include either **success** or an **error**. In the event of success, Travelgate will provide a response to the Seller by retrieving a breakdown of rate plans, encompassing details about the hotel, rates, and rooms.
