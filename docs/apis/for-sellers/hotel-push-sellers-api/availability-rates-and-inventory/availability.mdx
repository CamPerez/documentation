# Availability

## Availability Push

Method to **push** availability to the product.

### Request

The `HotelAvailNotif` request message pushes availabilities for the [inventory](./inventory) that has been previously set up.

[Añadir ejemplo]

### Response

After each request, Travelgate processes the data and provides a response to the Seller. Upon receiving a `HotelAvailNotif` request, Travelgate sends a corresponding `HotelAvailNotif` response. The response options include either **success** or an **error**.

**Success:**

```html
<HotelAvailNotifResponse xmlns="http://schemas.xmltravelgate.com/hubpush/provider/2012/10">
    <HotelAvailNotifResult>
        <Success xmlns="http://www.opentravel.org/OTA/2003/05"/>
    </HotelAvailNotifResult>
</HotelAvailNotifResponse>
```

**Error:**

```html
<HotelAvailNotifResponse xmlns="http://schemas.xmltravelgate.com/hubpush/provider/2012/10">
    <HotelAvailNotifResult>
        <Errors xmlns="http://www.opentravel.org/OTA/2003/05">
            <Error ShortText="AvailStatusMessages not found" Code="2"/>
        </Errors>
    </HotelAvailNotifResult>
</HotelAvailNotifResponse>
```

## Availability Retrieval

Method to **retrieve** the availability of the product.

### Request

The `HotelAvailRetrieve` request message is sent to retrieve a complete breakdown of availability, including details about the hotel, rate plans, and rooms.

[Añadir ejemplo]

### Response

After each request, Travelgate processes the data and provides a response to the Seller. Upon receiving a `HotelRatePlanRetrieve` request, Travelgate sends a corresponding `HotelRatePlanRetrieve` response. 

The response options include either **success** or an **error**. In the event of success, Travelgate will provide a response to the Seller by retrieving a breakdown of availability, including details about the hotel, rate plans, and rooms.

**Success:**

[Añadir ejemplo]

**Error:**

```html
<HotelAvailRetrieveResponse xmlns="http://schemas.xmltravelgate.com/hubpush/provider/2012/10">
    <HotelAvailRetrieveResult>
        <Errors xmlns="http://www.opentravel.org/OTA/2003/05">
            <Error ShortText="Incomplete AdditionalGuestAmount values" Code="7"/>
        </Errors>
    </HotelAvailRetrieveResult>
</HotelAvailRetrieveResponse>
```
