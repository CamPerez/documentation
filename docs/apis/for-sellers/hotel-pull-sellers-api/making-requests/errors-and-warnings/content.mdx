# Content

Below you will find a breakdown of the different errors that exist in the API to be returned to customers in the different content call responses. 
Clients are familiar with these types of errors and if they are well categorised they can help to highlight potential problems in property loading.

### Error List

| Code   | Description                                                                                                                                                          |
| -----  |----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 101  | System Exception (Exception not controlled or not classified as general exception)         |
| 102  | Supplier Error.                                                                                                                                |
| 103  | Too many requests to the supplier.                                                                                                             |
| 104  | Timeout (Timeout during the execution of an operation (look in the common attribute timeout )).                                            |
| 105  | Communication Error.                                                                                                                           |
| 206  | The Supplier doesn’t accept the dates RQ.                                                                                                      |
| 207  | The Supplier doesn’t accept the request RQ.                                                                                                    |
