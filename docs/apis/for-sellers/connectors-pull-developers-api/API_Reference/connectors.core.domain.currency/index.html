Currency

Namespace: Connectors.Core.Domain

Represents currencies as defined by the ISO 4217 standard.

public enum Currency

Inheritance Object ‚Üí ValueType ‚Üí Enum ‚Üí Currency
Implements IComparable, ISpanFormattable, IFormattable, IConvertible

Remarks:

ISO 4217 is the international standard for currency codes, providing unique three-letter codes (e.g., USD for US Dollar, EUR for Euro) and corresponding numeric codes for each currency. This enumeration includes all commonly used currencies supported by the Connectors framework.
