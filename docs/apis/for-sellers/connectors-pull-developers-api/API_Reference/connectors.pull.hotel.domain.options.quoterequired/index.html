QuoteRequired

Namespace: Connectors.Pull.Hotel.Domain.Options

Represents the requirements for quoting in the hotel domain.

public class QuoteRequired

Inheritance Object → QuoteRequired

Properties​

When​

Specifies the condition that determines when a quote is required.

public QuoteRequiredWhen When { get; set; }

Property Value​

QuoteRequiredWhen

Remarks:

The QuoteRequiredWhen enumeration defines the specific conditions under which a quote is necessary, such as always, on expiration, or in case of errors.

QuoteUrl​

The URL endpoint used to obtain a quote.

public string QuoteUrl { get; set; }

Property Value​

String

Remarks:

This property specifies the API endpoint or service URL where the quoting process can be initiated to retrieve up-to-date pricing and availability information.
