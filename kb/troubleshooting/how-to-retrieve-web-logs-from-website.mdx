# How to Retrieve Web Logs from TravelgateX Website

### Why are TravelgateX website logs important?💡
At times, our team may need to review your web logs to examine your connection to our website and address any potential issues.

### How can I obtain TravelgateX website logs?🛠️
1. Log onto TravelgateX website
1. **Click F12 or right-click and select "Inspect"**. Then, you will be able to visualize a new tab in your screen.
1. Once in that tab, elect "**Console**" secondary tab.

	```
	image
	```


1. Browse our website, some text will appear on the "**Inspect**" section.
1. Right-click the **Console** section and select "**Save as**", it will generate a log file.

	```
	image
	```

1. Select now the "**Network**" tab and click the "**Export HAR…**" button, it will generate another log file.

	```
	image
	```

1. Provide us with both log files and the e-mail that the user used to log in.

	```
	image
	```
